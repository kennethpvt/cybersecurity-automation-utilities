# Cybersecurity Automation Utilities

A robust collection of production-grade Python scripts designed to automate critical security tasks, including network reconnaissance, log analysis, credential assessment, and vulnerability scanning. This repository demonstrates practical scripting capabilities for Security Operations Center (SOC) engineering, penetration testing, and identity access management (IAM) compliance workflows.
